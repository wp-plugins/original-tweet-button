This plugin simply adds Twitter's official share button to the end of your posts.

== Description ==

Twitter has just released its own original Share button. And this plugin simply adds this button to the end of your posts.

To avoid the errornous issues, the plugin adds the button at the end of the posts ONLY in the individual pages of the posts (and pages). If you want to add the button wherever you want in the post, [check out the shortcode version of the plugin](http://wordpress.org/extend/plugins/original-tweet-button-shortcode/ "Original Tweet Button Shortcode").

The plugin works fine but it has no options or something else that you can tweak. However, it will be constantly developed.

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's all.

== Frequently Asked Questions ==

= This plugin adds the button to the end of the post but I want to add the button anywhere I want in the post. What do I do? =

[You install this plugin instead.](http://plugins.svn.wordpress.org/original-tweet-button-shortcode/ "Original Tweet Button Shortcode") It allows you to add the button with a simple shortcode: `[tweetthis]`
